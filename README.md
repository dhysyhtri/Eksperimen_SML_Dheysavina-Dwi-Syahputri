# Eksperimen_SML_Dheysavina-Dwi-Syahputri

Repositori ini berisi eksperimen **preprocessing data** untuk keperluan **Supervised Machine Learning (SML)**. Proyek ini bertujuan untuk membersihkan dan menyiapkan dataset mentah agar siap digunakan dalam model ML, serta mengotomasi proses preprocessing menggunakan Python dan GitHub Actions.

---

## Fitur
- Notebook interaktif untuk eksplorasi dan preprocessing dataset.
- Skrip Python untuk automasi preprocessing.
- Workflow GitHub Actions untuk menjalankan preprocessing otomatis saat ada commit.
- Dataset mentah dan hasil preprocessing tersimpan terpisah agar reproducible.
